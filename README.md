# Double Pendulum
A double pendulum simulation.

### Built With
* JavaScript
* HTML 5 Canvas

## Demo
Check the live demo here: https://patrickmargot.github.io/double-pendulum/

## Installation
1. Clone repository
```sh
git clone https://github.com/PatrickMargot/double-pendulum.git
```
2. Open `index.html` in your browser
